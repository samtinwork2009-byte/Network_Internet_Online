PACKET TRACER

Cisco Packet Tracer
網絡模擬平台

專業級網絡模擬器，提供真實的網絡設備配置、封包傳輸模擬及故障排除環境。 支援從基礎到進階的網絡技術實作，為網絡工程師及學習者提供接近實境的練習平台。

功能模組

核心

核心模擬引擎

完整的網絡模擬核心，支援多層次封包處理、路由協定模擬及網絡狀態監控。

  • 即時模式與模擬模式
  • 邏輯與實體工作區
  • 封包分析工具
  • 網絡拓撲設計
設備

網絡設備模擬

完整的 Cisco 設備模擬，支援各種路由器、交換器、防火牆及無線設備。

  • Cisco 路由器系列
  • 多層交換器
  • 無線網絡設備
  • 安全設備
協議

網絡協議模擬

完整的 TCP/IP 協議堆疊模擬，支援各種網絡協議配置與分析。

  • 路由協議 (OSPF, EIGRP)
  • 交換協議 (STP, VLAN)
  • 安全協議 (IPsec, SSL)
  • 應用層協議
安全

安全實驗室

完整的網絡安全實驗環境，支援防火牆配置、入侵檢測及安全策略實作。

  • 防火牆規則配置
  • VPN 隧道建立
  • 入侵檢測系統
  • 安全審計
自動化

網絡自動化

網絡自動化與編程環境，支援 Python、Ansible 及 REST API 整合。

  • Python 網絡自動化
  • Ansible Playbook
  • REST API 配置
  • 自動化監控
IoT

物聯網模擬

完整的物聯網生態系統模擬，支援各種感測器、致動器及智慧設備。

  • 環境感測器
  • 安全感測器
  • 智慧設備控制
  • 微控制器程式
此平台為專業級網絡模擬器，採用純前端技術實現所有網絡設備、協議及數據傳輸的模擬。 所有模擬操作均為虛擬環境中的練習，並非真實的網絡接駁。真正的網絡部署需要專業設備及實體佈線。

模擬器操作說明指南

關於此模擬器

Cisco Packet Tracer 網絡模擬器是一個專業級的網絡學習工具，讓您能夠設計、配置和測試虛擬網絡環境。通過這個平台，您可以學習路由器、交換器、防火牆等網絡設備的配置，以及各種網絡協議的運作原理。

1. 添加設備

操作步驟：

  1. 在左側工具面板中選擇設備類別
  2. 點擊或拖放設備到工作區
  3. 設備會自動獲得預設 IP 配置

注意事項：設備可在工作區內自由拖動調整位置

2. 連接設備

操作步驟：

  1. 選擇合適的連線類型（自動/直通線/交叉線等）
  2. 點擊第一個設備選擇連接端口
  3. 點擊第二個設備完成連線

線纜類型說明：

  • 自動連線：系統自動選擇合適線纜
  • 直通線：連接不同類型設備（PC-Switch）
  • 交叉線：連接相同類型設備（Switch-Switch）
  • Console線：用於設備管理配置

3. 配置設備

操作步驟：

  1. 點擊選中設備
  2. 在右側配置面板輸入參數
  3. 設定 IP 位址、子網路遮罩、預設閘道
  4. 點擊「套用配置」保存設定

CLI 配置：點擊「CLI終端」按鈕可開啟命令列介面

4. 模擬模式

即時模式：封包即時傳送，觀察網絡運作

模擬模式：逐步觀察封包傳遞過程，分析 OSI 各層

邏輯視圖：設計網絡拓撲結構

實體視圖：模擬設備實體擺放

5. 測試網絡

可用測試工具：

  • Ping 測試：測試設備間連通性
  • 路徑追蹤：查看封包傳輸路徑
  • 封包分析器：查看 OSI 各層封包詳情
  • 流量生成器：模擬網絡流量

重要提示

  • 每次修改配置後記得點擊「套用配置」

  • 定期匯出配置以防止數據丟失
  • 善用 CLI 終端進行進階配置
