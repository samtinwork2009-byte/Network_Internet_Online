模擬器操作說明指南 v2.0

關於 Cisco Packet Tracer 網絡模擬器

這是一個專業級網絡學習平台，提供六大模組：核心模擬（基礎拓撲設計）、設備管理（完整設備庫）、協議配置（OSPF/EIGRP/BGP）、安全實驗室（防火牆/VPN）、網絡自動化（Ansible/Terraform）及物聯網（IoT 感測器）。每個模組提供專屬的設備和工具，幫助您從入門到進階全面掌握網絡技術。

界面佈局

  • 左側面板：設備庫和連線工具
  • 中央區域：工作區畫布
  • 右側面板：設備配置表單
  • 底部欄：模式切換器
  • 頂部欄：進度指示和控制

快捷鍵參考

  • A 自動連線 | S 直通線
  • X 交叉線 | C Console
  • F 光纖 | R 串列線
  • P 封包分析 | L CLI終端
  • Del 刪除 | Esc 返回

步驟一：添加設備

兩種方式添加設備：

  1. 拖放方式：從左側設備庫拖曳設備到工作區指定位置
  2. 點擊方式：直接點擊設備圖標，自動添加到預設位置

設備類型說明：

  • 🔀 路由器：1941 / 2911 / 4321 - 網絡層轉發
  • 🔗 交換器：2960 / 3650 / 3850 - 數據鏈路層交換
  • 💻 終端：PC / Laptop / Server / IP Phone
  • 🛡️ 安全：ASA 防火牆 / IPS / VPN Gateway
  • 📡 無線：AP / WLC 無線控制器

步驟二：連接設備

連線流程（三步驟）：

  1. 在「連線工具」區選擇線纜類型
  2. 點擊第一個設備（設備會亮起綠色光暈）
  3. 點擊第二個設備完成連線

線纜類型選擇指南：

⚡ 自動連線系統自動判斷，推薦新手使用
🟢 直通線不同類型設備（PC ↔ Switch）
🟠 交叉線相同類型設備（Switch ↔ Switch）
🟣 Console管理連接（PC ↔ Router Console）
🔴 光纖高速長距離連接

步驟三：配置���備

基本配置流程：

  1. 點擊選中目標設備（設備邊框變藍）
  2. 在右側配置面板填寫參數：IP 位址、子網路遮罩、預設閘道
  3. 點擊「套用配置」按鈕保存設定

進階配置（CLI 終端）：

  • 點擊「CLI終端」打開命令列介面
  • 輸入 enable 進入特權模式
  • 輸入 configure terminal 進入配置模式
  • 常用命令：show ip interface brief、show running-config

步驟四：切換模式

四種工作模式（底部切換）：

  • ▶️ 即時模式：封包自動傳輸，觀察網絡實時運作
  • ⏯️ 模擬模式：逐步分析封包，查看 OSI 各層詳情
  • 📐 邏輯視圖：專注於網絡拓撲邏輯設計
  • 🏢 實體視圖：模擬設備實際物理擺放位置

步驟五：測試驗證

可用測試工具：

  • 🔍 Ping 測試：驗證設備間連通性，顯示延遲和封包丟失率
  • 🛤️ 路徑追蹤：查看封包經過的每一個跳點
  • 📊 封包分析器：深入分析 OSI 七層模型的封包詳情
  • 📈 效能監控：查看 CPU/記憶體/流量狀態
  • 🔧 診斷工具：自動檢測網絡問題並提供建議
